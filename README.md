# 🩺 Computer Vision–Based Disease Classification with LLM Clinical Guidance

This project presents an AI-powered healthcare assistance system that combines **Computer Vision** and **Large Language Models (LLMs)** to simulate a real-world diagnostic workflow.

The system first analyzes medical images using a deep learning–based computer vision model to classify diseases. Once a disease is predicted, the output is passed to a Large Language Model that generates structured, doctor-like clinical guidance.

The objective is not just disease detection, but providing meaningful medical context similar to what a healthcare professional would explain to a patient.

---

## 🔍 How It Works

### 1️⃣ Disease Classification (Computer Vision)

* Input: Medical image (e.g., X-ray)
* Model: CNN-based deep learning architecture (e.g., ResNet/EfficientNet)
* Output: Predicted disease label with confidence score

### 2️⃣ Clinical Guidance (LLM Integration)

The predicted disease is used as context for the LLM, which generates:

* Identified Disease
* Possible Causes / Medical Reasoning
* Recommended Diagnostic Tests
* Treatment Options
* Preventive Measures
* Things to Avoid
* Warning Signs for Immediate Medical Attention

This creates a structured and explainable diagnostic flow.
